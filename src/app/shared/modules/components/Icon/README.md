## Icon Component

### Usage
```html
<icon :icon="icon"/>
```

### Example
```html
<icon :icon="'EllipsisVerticalIcon'"/>
```

### Properties
| Name            | type                | Required | Description                   |
| --------------- | ------------------- | -------- | ----------------------------- |
| **icon**        | String              | true     | icon's name                   |
| **size**        | String              | false    | icon's size                   |
| **color**       | String              | false    | icon's color                  |
| **extraClass**  | String              | false    | icon's extra class            |
